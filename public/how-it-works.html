How It Works

Simple steps to build a trusted family record.

Start with one person, add relationships, and grow the tree with controlled access so everyone contributes safely.

01

Admin creates users

Roles define what each user can add or edit.

02

Link each user to a person

This sets the reference point for relationships.

03

Add family members

Users add children, spouses, parents, and more within their scope.

Relationship intelligence built in.

The system resolves relationships and supports Tamil family titles, helping everyone understand how people are connected.

Good data stays good

  • Ownership prevents accidental edits.
  • Admins can lock sensitive records.
  • Soft delete protects history.
